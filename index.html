This pages shows the published open-source packages from HKV Consultants.

Hkvsobekpy
HKVSOBEKPY is package for water-statistics and plausibility checker using his- and bui-files. Next to reading the his and bui-files, this package also contains some modules for applying statistics to obtain the return period for T10, T25, T50 and T100. This information is useful to compare against actual waterlevels.

Notebook: plausibiliteits validatie | Notebook: waterstand statistiek | Pypi | GitHub


Hkvfewspy
HKVFEWSPY is a Python wrapper for Delft-FEWS PI Web Services.

Test Notebooks | Pypi | GitHub


Hkvwaporpy
HKVWAPORPY is a Python package that connects to the FAO WaPOR webservice. This package provide options to retrieve the url where the raster information is stored on the server given the parameter, level and location.

Notebook: example usage | Pypi | GitHub
